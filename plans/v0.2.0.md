# v0.2.0

## Features

- `gcv branch` — interactive creator, inline create, `validate` subcommand
- Branch pattern validation against `branches` config in `.gc.json`
- `gcv init` — optional branches section and git hook installation (Husky or `core.hooksPath`)

## Config

Optional `branches` object:

```json
{
  "allowed": ["{type}/{description}", "{type}/{ticket}-{description}"],
  "types": ["feat", "fix", "hotfix", "release", "chore"]
}
```

## Hooks

- No Husky: `.gc/hooks/post-checkout` + `git config core.hooksPath .gc/hooks`
- With Husky: same hook file + delegation from `.husky/post-checkout`
